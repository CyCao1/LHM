# Legends_Monsters_and_Heroes

A rpg game that you play as hero to fight monsters.

Files: 
Main.java
Map.java
...


Platform: Mac OS

Run the game: 
1. From the Terminal, navigate to directory that contains .java file 
2. Run ```javac Main.java```
3. Run ```java Main```
